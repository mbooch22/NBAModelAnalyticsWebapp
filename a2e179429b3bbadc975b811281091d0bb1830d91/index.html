NBA Predictive Analysis Using Maching Learning

Using NBA statistics over the course of the last decade, a Machine Learning model was trained using AWS Amazon Sagemaker in order to predict statistics for future games.

  • Learn more

Web Scraping

Using a separate proprietary web scraping web app, NBA stats over the past decade were retreieved for every game played.

  • Learn more

Machine Learning

After retrieving and configuring all the advanced statistics from the Web Scraping app, the data was used to train a Machine Learning Model on AWS Amazon Sagemaker.

  • Learn more

Objectives

The predictions of the Model are used against the corresponding Vegas Odds. Various thresholds are used to determine the optimal chances of the model being more accurate than Sportsbooks.

  • Learn more

Results

See a breief overview of how to interact with the results of the NBA Predictive Machine Learning Model Below. Please click 'See Results' to get a detailed view of the Results.

Table View

The Results page will show a table of every game the NBA Predictive Model was used on. Proprietary information will remain hidden but you will be able to see: Teams, Scores, Vegas Odds, Whether the Model won or loss, etc.

Filters Available

Filters will be available to display different sets of data based on what is selected. Filter examples include: Dates, Wins, Losses, Type of Prediction/Bet, Whether the prediction was a threshold game, etc.

NBA Stats Realized

The goal of the Predictive Model is to have a winning percentage of at least 60% against the Vegas Odds. That would prove to be a profitable model. All projectections will be based on $100 wagers.

Eventually the model will have to be re-trained with the new data that is available. This is the best way to get the most accurate predictions. As of now, the last time the model was trained was February 25, 2021.

Charts

Multiple charts/graphs will be displayed to show the filtered data set. For example, Winning percentage will be shown in a pie chart.
